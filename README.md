# Kanban Board Backend

This is the backend of the **Kanban Board** application, built with **Django, Django REST Framework (DRF), and Python**. It handles **database operations, authentication, and authorization**.

## ✨ Features
- 🔒 **Authentication & Authorization**: Secure login and user management.
- 🗄 **Task & Contact API**: Full CRUD (Create, Read, Update, Delete) for tasks and contacts.
- 🔄 **RESTful API**: Endpoints for handling tasks, contacts, and users.
- 🛠 **Database Integration**: Stores tasks, contacts, and users in a database.

## 🛠️ Technologies Used
- **Django & Django REST Framework (DRF)**
- **Python**
